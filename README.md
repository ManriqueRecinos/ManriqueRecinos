## 🚀 Featured Projects

### 🏭 Real-Time Production Monitoring System *(Supertex)*
Built with **FastAPI**, **React 18 + Vite**, **WebSockets** and **TimescaleDB (PostgreSQL)**. Asynchronous architecture using `asyncpg` and `Pydantic`, with a frontend powered by **TailwindCSS**, **Recharts** and **Framer Motion**. Handles large volumes of time-series data for live production floor tracking.

### 📊 Production Dashboard *(Supertex – 6 months)*
Interactive real-time panel built with **Laravel**, **PHP**, **WebSockets** and **Pusher**. Includes dynamic charts and DataTables connected to live production processes to support operational decision-making.

### 📱 Employee Transport Control App *(Supertex)*
Mobile app in **React Native (Expo)** backed by **PostgreSQL via Supabase**. Scans employee ID cards to log transport records with real-time sync, authentication, and a modular, scalable architecture.

### 🐍 Python Services Optimization *(Supertex)*
Refactored a **Python** service that tracks module usage across Supertex's main application. Added new API endpoints, automated backups, and validated everything with **Postman**.

### 📦 Saas Inventory (My Startup)
A robust inventory and operational management system designed specifically for **MYPES**. Built with **Laravel**, **PostgreSQL**, and a custom SPA engine to provide a premium experience to small business owners.
